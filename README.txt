== Description ==

The simple way to integrate Spektrix with WordPress. No need for complicated API integration, the BoxOffice WP plugin is all that’s needed.

**Event Listing**
Effortlessly sync your website with Spektrix using our plugin, guaranteeing that your events are always up-to-date. Present them beautifully in a filterable and sortable grid, ensuring seamless navigation for your visitors.

**Event Pages**
With BoxOffice WP, each event is given its own search optimized page, complete with a comprehensive list of dates and convenient booking buttons. Boost your online visibility and make booking a breeze for your audience.

**Easy Installation and Configuration**
Setting up BoxOffice WP is straightforward. Simply install the plugin through WordPress and enter the URL of your Spektrix feed. Our user-friendly interface ensures a hassle-free configuration process.

**Streamlined Booking Process**
From event browsing to the final checkout, our plugin creates all the necessary pages to provide your users with a seamless booking experience. Streamline the entire process and enhance customer satisfaction.

**Gutenberg Blocks**
Take full control over your website’s layout and functionality with our dedicated Gutenberg blocks included in the BoxOffice WP plugin and theme. Empower your site editors with extensive customisation options.

**Highly Customisable**
BoxOffice WP offers a fully-functioning site right out of the box. However, we understand that each business is unique. That’s why we provide ample configuration options, allowing you to tailor the plugin and theme to suit your specific needs.

== Installation ==

Before installing, please ensure your WordPress website is configured to use the “Post Name” Permalink structure.

= Option 1: Using the WordPress dashboard =

1. Download the BoxOffice WP plugin ZIP file.
2. Log in to your WordPress website as an Administrator.
3. In the left-hand menu, click on “Plugins”.
4. Under “Plugins”, click the “Add New” sub menu.
5. Click the “Upload plugin” button.
6. Select the BoxOffice WP plugin ZIP file and click “Install now”.
7. After installation, you will find a menu item “BoxOffice WP”, click this to configure the plugin.

= Option 2: Using FTP =

1. Download the BoxOffice WP plugin ZIP file.
2. Upload ‘box-office-wp’ to the ‘/wp-content/plugins/’ directory.
3. Activate the plugin through the ‘Plugins’ menu in WordPress.
4. After installation, you will find a menu item “BoxOffice WP”, click this to configure the plugin.

== Frequently Asked Questions ==

= Do the free plugin and theme give me everything I need? =

Absolutely, install our free plugin and theme on a new instance of WordPress. Add the URL for your Spektrix feed and you can be up and running in 10 minutes.

= Can I upgrade to the Pro plugin at any time? =

We recommend starting with our free plugin, install it and see if it meets your needs, if not, BoxOffice WP Pro offers even richer features.

= Can I use the plugin with my existing website and theme? =

Of course, the BoxOffice WP plugin will work with any theme. When you activate the plugin it will create a handful of pages for you, if you’re operating an existing live website, we recommend installing on a Staging or Development version first.

= How can I get support? =

Please post any questions in the plugin support forum on wordpress.org. If you need further help, we can offer support packages to suit most needs, see details above.
